## 반응형 설계 특징

### 모바일 최적화
- 모든 카드형 레이아웃은 모바일에서 세로 스택으로 변환
- 터치 친화적인 버튼 크기 (최소 44px)
- 밈 생성기는 3열 → 1열로 변환
- 피드는 풀스크린 형태로 표시

### 태블릿 최적화
- 2열 레이아웃 활용
- 밈 생성기는 2열 (템플릿+편집, 미리보기)
- 적절한 여백과 패딩 조정

### 데스크톱 최적화
- 3열 레이아웃 최대 활용
- 고정 최대 너비 (max-w-6xl)
- 풍부한 여백과 그림자 효과
